# Superstore Sales Dashboard

## Project Overview

This project analyses sales performance using the Superstore dataset. The dashboard was built in Power BI to provide insights into sales, profit, product performance, and regional trends.

## Tools Used

- Power BI Desktop
- Excel
- DAX

## Dashboard Metrics

- Total Sales: 2.30M
- Total Profit: 285.99K
- Total Orders: 5K
- Total Quantity: 38K
- Average Order Value: 458.23
- Profit Margin: 12%

## Key Insights

1. Technology generated the highest sales revenue.
2. The West region produced the highest profit.
3. Sales increased during the final quarter of the year.
4. Top products contributed significantly to revenue.

## Recommendations

- Invest more in high-performing Technology products.
- Improve profitability in the Central region.
- Increase marketing during peak sales periods.
- Use product bundles to increase average order value.
